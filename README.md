# ML Smoothies Project

Welcome to the ML Smoothies Project! This project uses machine learning to generate a smoothie recipe curated to your specific cluster.

## Getting Started

To get started, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Run the `runme.py` script.

```bash
python runme.py
```

4. Enter your data when prompted.
5. The script will use our clustering model to generate a smoothie recipe tailored to your cluster.

## Requirements

- Python 3.x
- Required Python packages (listed in `requirements.txt`)

## Installation

Install the required packages using pip:

```bash
pip install -r requirements.txt
```

## Usage

Run the script and follow the prompts to enter your data:

```bash
python runme.py
```
